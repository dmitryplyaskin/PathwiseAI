# Breadcrumbs

## Описание

Компонент breadcrumbs для навигации по иерархии курсов, юнитов и уроков. Отображает реальные названия сущностей вместо ID.

## Файлы

- `frontend/src/shared/ui/breadcrumbs/types.ts` - типы данных
- `frontend/src/shared/ui/breadcrumbs/use-breadcrumbs.ts` - хук для генерации breadcrumbs
- `frontend/src/shared/ui/breadcrumbs/breadcrumbs.tsx` - компонент отображения
- `frontend/src/shared/ui/breadcrumbs/index.ts` - экспорты

## Функциональность

### Загрузка данных

Хук `useBreadcrumbs` автоматически загружает названия курсов и уроков через API:

- При переходе на страницу курса (`/courses/:id`) загружается название курса
- При переходе на страницу урока (`/courses/:id/units/:unitId/lessons/:lessonId` или `/courses/:id/lessons/:lessonId`) загружается название урока, а также кэшируются названия курса и юнита

### Кэширование

Названия сущностей кэшируются в памяти приложения для уменьшения количества запросов к API.

### Формирование breadcrumbs

Для разных маршрутов формируются следующие breadcrumbs:

- `/courses` → Главная > Курсы
- `/courses/:id` → Главная > Курсы > [Название курса]
- `/courses/:id/units/:unitId` → Главная > Курсы > [Название курса] > [Название юнита]
- `/courses/:id/units/:unitId/lessons/:lessonId` → Главная > Курсы > [Название курса] > [Название юнита] > [Название урока]
- `/courses/:id/lessons/:lessonId` → Главная > Курсы > [Название курса] > [Название урока]

Статические сегменты "units" и "lessons" пропускаются для улучшения UX.

### Отображение

- Последний элемент breadcrumbs отображается как текст (не ссылка)
- Остальные элементы - активные ссылки
- При наведении на ссылку меняется цвет и добавляется анимация

## API Endpoints

Используются следующие endpoints:

- `GET /courses/:id` - получение информации о курсе
- `GET /courses/lessons/:id` - получение информации об уроке

## Использование

Компонент автоматически используется в `Layout` для всех страниц, кроме главной и страниц авторизации.
