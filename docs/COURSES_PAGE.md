# Страница курсов

## Описание

Страница `/courses` отображает список всех доступных курсов пользователя с дополнительной информацией о последних уроках каждого курса.

## Функциональность

### Отображение курсов

- Список всех курсов пользователя в виде карточек
- Фильтрация по статусу (все, в процессе, завершенные, новые)
- Фильтрация по категории и сложности
- Поиск по названию, описанию и тегам
- Сортировка по рейтингу, прогрессу, популярности или названию
- Переключение между видами сетки и списка

### Последние уроки

- Отображение последних 3 уроков для каждого курса
- Статус каждого урока (не начат, изучается, освоен)
- Время чтения урока
- Индикатор загрузки при получении данных

### Навигация

- Клик по карточке курса ведет на страницу курса (`/courses/:id`)
- Сохранение всех фильтров и настроек просмотра

## Техническая реализация

### Backend

#### Новый эндпоинт

- `GET /courses/:id/lessons` - получение уроков конкретного курса
- Проверка доступа к курсу через AccessControlService
- Сортировка уроков по дате создания (DESC)

#### Файлы

- `src/modules/courses/controllers/courses.controller.ts` - добавлен метод `getCourseLessons`
- `src/modules/courses/services/lessons.service.ts` - добавлен метод `findLessonsByCourseId`

### Frontend

#### Новые компоненты

- `frontend/src/features/course-card/ui/CourseCardWithLessons.tsx` - карточка курса с уроками

#### Обновленные компоненты

- `frontend/src/widgets/courses-list/ui/CoursesList.tsx` - использует новый компонент карточки
- `frontend/src/shared/api/courses/api.ts` - добавлен метод `getCourseLessons`

#### API интеграция

- Загрузка уроков происходит асинхронно для каждого курса
- Обработка состояний загрузки и ошибок
- Кэширование результатов

## Использование

Страница доступна по маршруту `/courses` и автоматически загружает:

1. Список курсов пользователя
2. Последние 3 урока для каждого курса
3. Статистику по прогрессу и рейтингам

Все данные обновляются при изменении фильтров или поискового запроса.
