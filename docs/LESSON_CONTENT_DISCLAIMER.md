# Дисклеймер для контента урока (AI-generated)

## Что изменено

Внизу контента урока (после рендера markdown) отображается дисклеймер о том, что материал сформирован с помощью ИИ и может содержать ошибки/неточности.

## Где реализовано

- `frontend/src/widgets/lesson/ui/lesson_content.tsx`

## Текст дисклеймера

Текст задаётся локально в компоненте и выводится мелким шрифтом (`Typography variant="caption"`) сразу после блока `MarkdownRenderer`.
